Question 1: Recipe App

Create a visually appealing recipe app using Ionic-React with the following structure:

1. Welcome Screen:

  - Display a welcome message to users interested in exploring new recipes.
  - Include a "Get Started" button to navigate to the recipe catalog.

2. Recipe Catalog:

  - On tapping "Get Started" on the Welcome Screen, users should be directed to the Recipe Catalog.
  - Display a collection of recipes using cards with images fetched from a stock image website.
  - Include a side menu with options to navigate back to the Welcome Screen or to a Settings screen for personalized preferences.
